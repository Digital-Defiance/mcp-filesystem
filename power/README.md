# ACS Filesystem Manager Kiro Power

This directory contains the Kiro Power definition for the ACS Filesystem Manager MCP server.

## What is a Kiro Power?

A Kiro Power packages MCP servers, documentation, and workflows into a reusable capability that can be installed in Kiro IDE.

## Files

- `POWER.md` - Power metadata and documentation
- `mcp.json` - MCP server configuration
- `steering/` - Workflow guides and best practices

## Installation

Users can install this power in Kiro IDE by:

1. Browsing the Kiro Powers registry
2. Searching for "ACS Filesystem Manager"
3. Clicking "Install"

The power will automatically configure the MCP server and make filesystem tools available to AI agents.
